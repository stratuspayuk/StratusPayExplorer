API Documentation

Welcome to the StratusPayAPI documentation. Here you'll find all the information you need to integrate with our payroll calculation engine.

POST /api/v1/payroll/calculate

This is the primary endpoint for calculating payroll. It takes a detailed request object and returns a comprehensive breakdown of pay, deductions, and contributions.

Request Parameters

Parameter Type Description
annualGrossSalarynumberThe employee's annual gross salary.
taxCodestringThe employee's tax code (e.g., "1257L").
payFrequencystringThe pay frequency ("Monthly", "Weekly", "Annual").
payPeriodDatestringThe date of the pay period in YYYY-MM-DD format.
employeeContributionPercentagenumberThe employee's pension contribution percentage.
employerContributionPercentagenumberThe employer's pension contribution percentage.
pensionTypestringThe pension type ("ReliefAtSource", "SalarySacrifice", "NetPayArrangement", "None").
isPlan1booleanWhether the employee has a Plan 1 student loan.
isPlan2booleanWhether the employee has a Plan 2 student loan.
isPlan4booleanWhether the employee has a Plan 4 student loan.
isPlan5booleanWhether the employee has a Plan 5 student loan.
isPostgraduatebooleanWhether the employee has a postgraduate loan.
isScottishResidentbooleanWhether the employee is a Scottish resident.
isBlindPersonbooleanWhether the employee is registered blind.
isMarriageAllowanceTransferorbooleanWhether the employee is transferring the marriage allowance.
isMarriageAllowanceRecipientbooleanWhether the employee is receiving the marriage allowance.
